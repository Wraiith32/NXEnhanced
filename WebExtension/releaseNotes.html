NX Enhanced 5.0 Release Notes

NextDNS replaced their API with a new one that is structured differently, which in turn broke NX Enhanced. This update adapts NXE to this new API, with some features that had to be rewritten, and also comes with some nice changes and improvements.

General:
  • Adapted the features in the Security, Allowlist, Denylist, Logs and Settings pages to the new API;
Security page:
  • Rewritten the "Add all TLDs" feature. Now all TLDs are added almost instantly. No more having to wait 30 minutes!
Allowlist/Denylist page:
  • When importing a list of domains and an error occurs with one of the domains, now it's specified what domain caused the error;
Logs page:
  • The "Other devices" feature is much faster and lighter than before;
  • The Allow/Deny popup is dismissed a bit faster when allowing/denying. When hiding a domain, it's now twice as fast;
  • Now the domain filter doesn't require a dot at the beginning of the domain. It now matches both the exact domain *and* subdomains, instead of just the subdomains;
  • Fixed bug where the logs stop loading when the domain filter is disabled then enabled and almost all log entries got filtered.

Important

The configuration import feature in this version is not compatible with the previous versions. If you have a backup of your config saved somewhere, make sure to export again and replace it with the new one.
